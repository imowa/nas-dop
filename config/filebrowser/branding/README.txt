FileBrowser custom branding (mobile-first, GDrive-like). Most customers open share links on their phone.

- custom.css  = theme (already included). Edit to tweak colors/layout.
- img/        = optional: put logo.svg here (in img/).
- img/icons/  = optional: favicon.ico + PNGs (16x16, 32x32). Use https://realfavicongenerator.net/
  Clear browser cache after changes.

In FileBrowser: Settings → Global Settings → Branding directory path = /branding
